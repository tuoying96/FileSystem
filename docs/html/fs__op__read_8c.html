read - read fuse_parser_data from an open file.

Should return exactly the number of bytes requested, except:

  • if offset >= file len, return 0
  • if offset+len > file len, return bytes from offset to EOF
  • on error, return <0

Errors: -ENOENT - file does not exist -ENOTDIR - component of path not a directory -EISDIR - file is a directory -EIO - error reading block

Parameters
paththe path to the file
bufthe read buffer
lenthe number of bytes to read
offsetto start reading at
fifuse file info
Returns
number of bytes actually read if successful, or -error number
